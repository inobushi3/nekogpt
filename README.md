# NekoGPT Free

NekoGPT Free is the free Windows edition of NekoGPT, an AI virtual companion focused on local-first chat, voice interaction, and a Live2D desktop experience.

This repository is used only to publish the public installer. The application source code is not distributed here.

## What Is Included In Free

- Chat with supported AI providers.
- Ollama local model setup for offline/local AI use.
- Google AI provider support.
- Voice conversation with local microphone transcription.
- Kokoro and Neko Local TTS support.
- Default personas.
- Default Live2D companion experience.

## Install Notes

- Platform: Windows x64.
- Download the `.exe` from the Release page.
- Run the installer and open NekoGPT Free.
- On first setup, local AI and voice dependencies may be prepared automatically.
- Ollama models are stored under `Documents\NekoGPT Free\models`.
